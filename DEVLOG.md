# Day 5 Dev Log – AI Spend Audit Tool

## Project Progress
Today I completed the core functionality of my **AI Spend Audit Tool** using **Next.js, React, TypeScript, and Tailwind CSS**.

## Features Implemented

### 1. AI Pricing Dashboard
Created pricing cards for major AI tools:
- ChatGPT
- Claude
- GitHub Copilot
- Gemini

Each tool includes:
- Multiple subscription plans
- Monthly pricing
- Key features
- Color-coded pricing display


### 2. AI Spend Audit Form
Built an interactive audit form with:
- User Type selector (Student / Professional / Business)
- AI Tool selector
- Monthly Spend input
- Team Size input
- Audit My Spending button


### 3. Spending Analysis Logic
Implemented audit calculations for:
- Recommended monthly spending
- Extra spending / saved amount
- Yearly cost difference
- Per person cost calculation
- Spending grade evaluation


### 4. Recommendation System
Added dynamic feedback messages based on spending:
- Optimized spending
- Overpaying warning
- Student savings suggestions
- Business usage recommendations


### 5. Dynamic Result Dashboard
After submitting the form, users can view:
- Monthly Status
- Yearly Status
- Recommended Spending
- Per Person Cost
- Spending Grade
- Personalized Message


### 6. UI/UX Improvements
Improved interface using Tailwind CSS:
- Gradient dark theme
- Responsive two-column layout
- Glassmorphism cards
- Hover button effects
- Color-coded status indicators


## Technologies Used
- Next.js
- React Hooks (`useState`)
- TypeScript
- Tailwind CSS


## Challenges Faced
- Managing multiple state variables
- Implementing conditional pricing logic
- Handling dynamic UI updates after form submission
- Designing responsive layout


## What I Learned
Today I improved my understanding of:
- React state management
- Form handling in React
- Conditional rendering
- Dynamic calculations
- Building interactive dashboards
